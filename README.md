# sb7-storybrand

> 把 Donald Miller《Building a StoryBrand》(2017) 的 SB7 七元素框架，提炼成**商业咨询方案 / 品牌话语 / 销售文案**的诊断与重构 skill。

**核心信条：客户是英雄，咨询师 / 品牌是向导。**

凡是文案以"我们成立于 2015 / 团队来自 BAT / 资深团队丰富经验"开头 — 错。
重写到 → "你（客户）想要什么 → 你卡在哪 → 我（向导）怎么带你穿越"。

如果你只能记住一件事：**记住这一条。**

---

## 为什么需要这个 skill

90% 的咨询方案、提案 PPT、官网首屏死法都是同一个：**品牌当英雄、客户当观众**。结果就是：
- 客户读完不知道方案在卖什么
- 看完没冲动行动
- 和你的竞品在描述上无差别
- 5 秒内回答不出 "怎么让我变好"

SB7 是 Donald Miller 在好莱坞编剧理论 + 2 万家企业咨询里淬出来的一套**叙事公式**。本 skill 把它从概念书变成可调用的诊断-重构操作器。

---

## 何时用 / 何时不用

### ✅ 用本 skill

- 写 / 改商业咨询方案、提案 PPT、销售话术
- 写 / 改产品介绍、官网首屏、邮件 outreach
- 客户反馈"看不懂、卖不出去、差异化不够"
- 对外传播话语打磨（公众号导流文、广告 landing page、Pitch Deck）

### ❌ 不要用

- 内部流程文档、技术架构文档
- 与客户传播无关的内部材料
- 私域裂变模板（那是另一套打法）
- 复杂 B2B 多层决策链的销售手册（SB7 主要面向单决策者沟通）

## 必备输入三件套（缺则 skill 会主动追问）

### (a) 原始方案 / 文案
粘贴文本 / 文件路径 / 链接都可以。

### (b) 目标客户画像
不是"中小企业"这种宽泛说法，而是：
- **是谁**：行业 + 阶段 + 角色（例："准备 A 轮的早期 SaaS 创始人"）
- **想要什么**：一个清晰、有冲突的目标
- **怕什么**：阻止他们行动的内心戏

### (c) 业务实际情况
特别关键的是**权威凭证**：
- 服务过多少类似客户？
- 有什么具体数字结果？（例："平均把 CAC 砍 47%"）
- 有什么第三方背书？
- 有什么可以放上去的退款 / 保证条款？

⚠️ **这块不给具体数字，重构出来的方案就站不住**。第一次跑 skill 时如果还没想好，让 skill 帮你把缺位标出来再回头补。

## 你会得到什么（输出 6 块）

### 1. SB7 诊断表
13 个细项标 ✅/⚠️/❌，让你先看清原方案缺什么、弱在哪：

| 元素 | 状态 | 原文/缺位 | 优化方向 |
|---|---|---|---|
| 人物画像 | ❌ | "中小企业"太宽 | 收窄到具体阶段+角色 |
| 外在问题 | ⚠️ | 只一句话 | 加数字 |
| ... | | | |

❌ 如果 ≥ 7 项 = **这不是优化是重写**。

### 2. BrandScript 一页骨架
七元素压缩到一页，方便贴在墙上对着写：

```
人物：[客户画像 + 想要]
问题：
  外在 / 内在 / 哲学 / 反派
向导：
  同理心 / 权威
计划：
  流程 / 协议
CTA：
  直接 / 过渡
失败代价 / 成功画面
```

### 3. 一句话脚本
`帮 [客户] 解决 [问题]，让他们 [成功画面]。`

例：「帮准备 A 轮的早期 SaaS 创始人在 90 天内把 CAC 砍掉 30% 以上，让你不再为了拉客户烧光融资。」

放在自我介绍、公众号 banner、官网 hero、合作邮件第一句 — **一句话定位的最强压缩格式**。

### 4. 重构后的方案大纲
按 SB7 叙事顺序重排（不是补内容，是把原方案的章节顺序改了）：

```
§1 钩子段（外在问题 + 反派）       ~10%
§2 利害段（失败代价 + 成功画面）   ~15%
§3 向导段（同理心 + 权威）         ~20%
§4 计划段（流程 + 协议）           ~30%
§5 CTA 段（直接 + 过渡）           ~15%
附录                               ~10%
```

⚠️ 关键：**前 30% 决定客户读不读完**。所有钩子、利害、向导都必须挤在前 30%。

### 5. 5 句即用文案
5 个**可直接粘贴**的强力句子，分别覆盖：钩子句 / CTA 句 / 对比句 / 保证句 / 画面句。

例：
- 对比句：「99 元的增长课告诉你方法论，我们承诺数字 — 不达 30% CAC 降幅，全额退款。」
- 保证句：「过去 5 次执行这条退款条款，我们赔出过 3 次。我们不怕赔，怕的是接错客户。」

### 6. Grunt Test 结果
检查方案能否在 **5 秒内**回答这三问：
- 你提供什么？
- 怎么让我的生活变好？
- 我要做什么才能拿到？

任何一项答不出 = 方案没过关。

## SB7 七元素速览

| # | 元素 | 关键问题 | 一句话标准 |
|---|---|---|---|
| 1 | **人物**（Character） | 客户是谁，想要什么 | 画像 + 一个清晰、有冲突的目标 |
| 2 | **问题**（Problem） | 卡在哪 | 三层 + 反派：外在 / 内在 / 哲学 / 反派 |
| 3 | **向导**（Guide） | 凭啥相信你 | 同理心 + 权威，缺一不可 |
| 4 | **计划**（Plan） | 怎么干 | 流程计划（步骤）+ 协议计划（消除恐惧） |
| 5 | **CTA**（Action） | 现在做什么 | 直接 CTA + 过渡 CTA，缺一不可 |
| 6 | **失败代价**（Stakes） | 不行动会怎样 | 时间窗 + 数字 + 谁替代你 |
| 7 | **成功画面**（Success） | 行动后会怎样 | 数字 + 场景 + 情绪 |

## 真实例子（节选 SaaS 增长咨询提案）

### Before（典型死法）

> # 助力企业增长的专业咨询
>
> 我们是一支资深的增长咨询团队，由前 BAT 增长团队成员组成，拥有丰富的行业经验。我们专注于为中小企业提供全方位的增长解决方案...
>
> ## 我们的优势
> - 资深团队 - 丰富经验 - 专业服务
>
> 期待与您合作！

诊断：13 项中 ❌ 9 项、⚠️ 2 项、✅ 0 项 — **不是优化是重写**。

### After（SB7 重构）

**§1 钩子段**：
> 你已经在投放上烧了 6 个月，每月 50 万预算，但合格线索还是不到 30 个。
> 不是你不努力 — 是那些把"增长黑客"包装成 99 元课程的人，用"私域、漏斗、ABM"几个名词，让你以为问题是你掌握得不够多。

**§2 利害段**：
> **如果继续按现状跑 6 个月**：再烧 300 万，CAC 维持 ¥1,500+，同期已有 2 家竞品从你的目标客户名单里签下 18 家。
> **如果重构话语层**：90 天后 CAC 降到 ¥600，每月稳定 80+ 合格线索。下次和投资人聊增长时，你拿出来的不是借口，是一份可复制的剧本。

**§3 向导段**：
> 我知道你已经看完 50 篇增长黑客文章，每读一篇都更焦虑而不是更清楚。
> 过去 18 个月，我们帮 23 家早期 SaaS 重构话语层，平均把 CAC 砍 47%。其中 3 家完成 A 轮。
> 不达成 30% CAC 降幅就全额退款 — 这条条款过去 5 次执行里，我们赔出过 3 次。

**§4 计划段**（节选）：
> - 第 1 周：增长漏斗诊断 → 《卡点报告》
> - 第 2 周：话语层重构 → BrandScript + 5 套对照素材
> - 第 5 周：定稿可复制的投放剧本
> 协议：不达 30% 降幅退款 / 双周对齐数据全透明 / 主咨询师全程在场

**§5 CTA**：
> 👉 预约 30 分钟首次诊断（免费）—— 直接看你的数据
> 📥 下载《SaaS 增长漏斗诊断手册》PDF —— 没准备好沟通先用这本自检
> ⚠️ 本季度还剩 2 个名额。

完整 before/after 工作样例见 [examples/before-after-saas.md](examples/before-after-saas.md)。

## 一定不要犯的 8 个错（Anti-Patterns）

1. **品牌当英雄** — "我们成立于 2015..." 开头 = 死。
2. **三层问题只写外在** — 没内在/哲学，文案就是产品说明书。
3. **CTA 是"如果你感兴趣可联系我们"** — 这不是 CTA，是放弃。
4. **失败代价写软** — "可能错过机会" = 没说。要数字、时间窗、谁替代你。
5. **成功画面只写功能不写感受** — 客户买的不是功能，是变成什么样的自己。
6. **全文没反派** — 没反派 = 没张力 = 客户感觉不到为什么要买。
7. **同理心 + 权威各只一句** — 这是被信任的全部基础，要展开。
8. **过度抽象** — "赋能、增效、转型、卓越、专业"全是噪音词，删掉。

## 与其他 skill 协作

本 skill 设计为**内容武器层**，可以和这些上下游链：

| Skill | 关系 | 怎么用 |
|---|---|---|
| `ai-brand-positioning` | 上游 | 先做品牌身份/视觉调性，再用本 skill 做客户向话语 |
| `wechat-writer` | 下游 | 把 BrandScript 当公众号长文骨架（钩子段→主体→CTA+代价+画面） |
| `ai-viral-content-engine` | 下游 | 把"5 句即用文案"喂给短视频脚本 |
| `agency-marketing` | 聚合层 | 作为子能力被调用 |
| `os-yuanli` | 上层治理 | os-yuanli 决定"该不该做这个咨询/走哪条赢"，本 skill 决定"话语怎么打" |

例：写一篇商业咨询导流公众号文 →
1. `os-yuanli` 判断该不该写
2. `ai-brand-positioning` 定品牌调性
3. `sb7-storybrand` 出 BrandScript + 5 句即用文案
4. `wechat-writer` 把 SB7 输出展开成长文
5. `content-quality-gate` 终审
